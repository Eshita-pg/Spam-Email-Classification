Spam email classification using Support Vector Machine:
Using SVM to classify emails into spam or
non-spam categories and reporting the classification accuracy for various
SVM parameters and kernel functions.








Directions for running:

In windows terminal give command [this command will work if required packages are already installed]

 python 7.py





OR 



To check remotely, open 7.ipynb in Google Colab/Jupyter Notebook and then run accordingly.
